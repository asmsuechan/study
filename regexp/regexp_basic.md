# [正規表現の基礎]

### urlの最後のスラッシュ以下
```
/[^\/]+$/

# rubyでの例
url = "http://example.com/hoge/fuga/123"
id = url.slice(/[^\/]+$/)

puts id
=> "123"
```

### +と*と?の違い
+は1文字以上で*は0文字以上、?は0文字か1文字

```
# マッチするのは"hoge"または""
/(hoge)?/
```
